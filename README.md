# 🐄 Cow Breed Advisor - Sustainable & Scalable Cow Breeding Awareness Platform  

## 🚀 About the Project  

*Cow Breed Advisor* is an innovative platform developed for the *Google Solutions Hackathon* to promote sustainable and scalable cow breeding practices. Our project leverages modern technology to spread awareness and assist farmers in making informed breeding decisions.  

With *GeminiAI integration*, we provide AI-driven compatibility analysis to help farmers select the best breeding options, ensuring healthier cattle and improved productivity.  

---

## 🛠 Tech Stack  

We have built this project using cutting-edge technologies to ensure a seamless and engaging user experience:  

### 🌐 Frontend  
- *React.js* ⚛ - For a dynamic and responsive user interface  
- *JavaScript (ES6+)* 🟨 - Core scripting language  
- *Tailwind CSS* 🎨 - For sleek and modern UI styling  

### 🧠 AI Integration  
- *GeminiAI* 🤖 - Enables intelligent breeding compatibility analysis  

### 🗄 Backend & Database  
- *MongoDB* 🟢 - NoSQL database for efficient data storage  

### 📦 Additional Libraries & Packages  
- *Framer Motion* 🎥 - Smooth animations  
- *Heroicons / React Icons* 🔥 - Aesthetic icons for better UI/UX  
- *Axios* 🔗 - API handling  

---

## 🌟 Key Features  

✅ *AI-Powered Breeding Compatibility Analysis* – Get AI-driven suggestions for optimal breeding choices.  

✅ *Compiled News Page* 📰 – Aggregates all relevant news about cow breeding and livestock farming from a reliable API source, keeping users updated with industry trends.  

✅ *Comprehensive Information Page* 📚 – A single repository of:  
  - Government initiatives 🏛 related to cattle breeding and dairy farming.  
  - Research papers 📑 to provide scientific insights.  
  - Learning resources 🎓 to educate farmers and stakeholders about best breeding practices.  

✅ *User-Friendly Interface* 🎨 – A clean, modern UI with smooth animations for a better user experience.
